آموزش سریع پایتون — نمونه‌ها

چند مثال کوتاه برای شروع: متغیر، لیست، حلقه، تابع و اجرای ساده

مثال‌ها (قابل کپی)

۱) متغیر و چاپ 
name = "Mahdi"
age = 21
print(f"سلام {name} — شما {age} ساله‌اید")
۲) لیست و حلقه 
fruits = ["سیب", "موز", "پرتقال"]
for i, f in enumerate(fruits, 1):
    print(i, "-", f)
۳) تابع ساده 
def greet(name):
    return "سلام " + name

print(greet("دنیا"))
۴) اجرا در مرورگر (ترجمه‌شده برای نمایش) 
# برای اجرای واقعی، از مفسر پایتون استفاده کنید.
# مثال: ذخیره در file.py و اجرای `python file.py`

نکات سریع

  • برای شروع، Python 3.10+ نصب کن.
  • کدها را در فایل با پسوند .py ذخیره و با دستور python file.py اجرا کن.
  • از ویرایشگرهایی مثل VSCode یا PyCharm استفاده کن تا راحت‌تر دیباگ کنی.
تصویر بالا هر بار تازه (تصادفی) بارگیری می‌شود. برای اجرای واقعی کدها از مفسر پایتون استفاده کنید.
